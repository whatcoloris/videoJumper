# videoJumper
taking in microphone input, using it to module playback of an mp4, requires pocketsphinx

## To Use

### get a virtual environment going
`python3 -m venv env`
`source env/bin/activate`

### get requirements
`pip install -r requirements.txt`

### download video to folder with python script

### run script with video as argument
`videoJumper.py XXXXX`
